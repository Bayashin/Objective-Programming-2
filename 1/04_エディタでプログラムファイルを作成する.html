エディタでプログラムファイルを作成する

まえおき

オブジェクト指向プログラミング及び演習２では標準のエディタとしてVS Code（ビジュアルスタジオコード）を利用します．

Python文法のシンタックスハイライトやdocstringの表示など便利な機能が備わっていますがあくまでもエディタとして紹介します

VS Codeのインストール

  1. VisualStudio Codeをインストールbrew cask install visual-studio-code

  2. ターミナルで講義フォルダを開くcd $HOME/work/oop2

  3. VS Codeの起動コマンドを実行 code . （codeの後ろに半角スペースとドットが必要です）

    • 起動しない場合

      • 一度VS Codeをアプリランチャーから起動してVS Code内で⌘+Shift+Pを入力して上部パレットに>Shellと入力

      • Shell Command: Install 'code' command in PATHを選択

      • VS Codeを閉じてターミナルに戻り，再度code .と入力

  4. 新しくファイルを作成

    • VS Code左のメニューからファイル形のアイコン（Explorer ⌘+Shift+E）を選択

    • oop2ディレクトリへマウスカーソルを合わせると「ファイル作成，ディレクトリ作成，リロード，全て展開」のアイコンが見えるのでファイル作成アイコンをクリック

    • ファイル名に”lecture01_01.py”と入力する（下の図のようにファイル拡張子が見えるように設定されていない人はTAを呼ぶこと）

    • 初めて.pyファイルを開くときはPython言語用のエクステンションをインストールする（インストールをクリック）

  5. プログラムを実行

    • Pythonのバージョンを確認する（VS Code画面左下のPythonのバージョンが3.9でなければ画面左下をクリックしてバージョン3.9を選択する）

    • 実行したいプログラムをlecture01_01.pyに書き込む

    • 画面右上にプログラム実行ボタンがあるのでクリックするとターミナルが開き実行される

  6. おまけ

    • 行頭スペースを可視化するにはsettings.jsonに"editor.renderWhitespace": "boundary"を追加する（わからなかったらTAへ）

解説: メイン関数の代わりにif __name__ == '__main__':

  • if __name__ == '__main__':の部分は，このファイルが実行された時だけif文の処理が実行される．（別のファイルからこのプログラムを呼び出した時は実行されません．if文の中がメイン関数の処理だと考えて下さい．）
