PythonでGUIアプリケーションを作る基礎を学ぶ

  • GUIアプリ作成のためのPySide6(PyQt6)モジュールの利用

    • PyQt6公式Doc：https://www.riverbankcomputing.com/static/Docs/PyQt6/

    • Qt for Pythonのチュートリアルと例：https://doc.qt.io/qtforpython/contents.html

  • インストール方法

    • pip install pyside6

  • インストール確認用サンプルプログラム(hello_pyside.py)

import sys
from PySide6.QtWidgets import QApplication, QLabel

app = QApplication(sys.argv)
label = QLabel("Hello World!")
label.show()
app.exec()
